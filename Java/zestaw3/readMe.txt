# Currency Converter

## Opis
Prosty kalkulator przeliczający kwoty pieniężne między polskim złotym (PLN) a trzema innymi walutami: dolarem amerykańskim (USD), euro (EUR) i funtem brytyjskim (GBP). Użytkownik może wybrać jedną z tych walut i podać kwotę w złotych do przeliczenia.

## Instrukcje użytkowania
1. Uruchom program.
2. Wybierz walutę, dla której chcesz przeliczyć kwotę, podając odpowiedni numer (1 dla USD, 2 dla EUR, 3 dla GBP).
3. Podaj kwotę w złotych do przeliczenia.
4. Program wyświetli przeliczoną kwotę w wybranej walucie.

## Obsługa błędów
- W przypadku błędu wejścia/wyjścia, program wyświetli komunikat "Błąd operacji wejścia/wyjścia."
- W przypadku nieprawidłowego formatu liczby, program wyświetli komunikat "Nieprawidłowy format liczby."

## Dodatkowe uwagi
- Kursy wymiany są zdefiniowane dla dolara amerykańskiego (USD), euro (EUR) i funta brytyjskiego (GBP).
- Wyniki są zaokrąglane do dwóch miejsc po przecinku.

## Możliwe ulepszenia
- Rozważ dodanie obsługi innych walut i ich kursów.
- Dodaj funkcję umożliwiającą użytkownikowi ponowne przeliczenie kwoty bez konieczności ponownego uruchamiania programu.
