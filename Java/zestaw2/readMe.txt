# README

## Zadania z programowania obiektowego w języku Java

W tym repozytorium znajdziesz kod źródłowy, który obejmuje implementacje kilku klas w języku Java, przedstawiających koncepty programowania obiektowego. Poniżej znajdziesz krótkie omówienie każdej z klasy wraz z przykładami użycia.

### 1. Ulamek

Klasa `Ulamek` reprezentuje ułamek matematyczny. Posiada metody do dodawania, odejmowania, mnożenia, dzielenia, odwracania i skracania ułamków. Przykłady użycia znajdują się w klasie `Main`.

### 2. LiczbaZespolona

Klasa `LiczbaZespolona` modeluje liczby zespolone. Udostępnia metody do dodawania, odejmowania, mnożenia i dzielenia liczb zespolonych. Przykłady użycia znajdziesz w klasie `Main`.

### 3. Wektor

Klasa `Wektor` reprezentuje trójwymiarowy wektor. Zawiera metody do dodawania, odejmowania, obliczania iloczynu skalarnego, iloczynu wektorowego i obliczania długości wektora. Przykłady użycia są dostępne w klasie `Main`.

### 4. WielomianKwadratowy

Klasa `WielomianKwadratowy` modeluje wielomian kwadratowy. Udostępnia metody do dodawania, mnożenia oraz znajdowania miejsc zerowych. Przykłady użycia są również dostępne w klasie `Main`.

### Jak uruchomić przykłady

1. Sklonuj to repozytorium na swój lokalny komputer.
2. Uruchom klasę `Main` w środowisku programistycznym obsługującym język Java.

Mam nadzieję, że ten kod i krótkie readme pomogą Ci zrozumieć i używać podstawowych konceptów programowania obiektowego w języku Java. Jeśli masz pytania lub potrzebujesz dodatkowej pomocy, śmiało pytaj!
