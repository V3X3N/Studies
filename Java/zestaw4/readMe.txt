# Figury Geometryczne

Projekt ten implementuje hierarchię klas reprezentujących różne figury geometryczne, takie jak kwadrat, okrąg, prostokąt i trójkąt. Wykorzystuje on koncepcję dziedziczenia i polimorfizmu, umożliwiając operacje na figurach ogólnego typu.

## Struktura Katalogów

- `Figura`: Abstrakcyjna klasa bazowa definiująca interfejs dla figur geometrycznych.
- `Kwadrat`: Klasa reprezentująca kwadrat.
- `Okrag`: Klasa reprezentująca okrąg.
- `Prostokat`: Klasa reprezentująca prostokąt.
- `Trojkat`: Klasa reprezentująca trójkąt.
- `Program`: Główna klasa zawierająca metodę `main` do testowania różnych figur.

## Jak Uruchomić

1. Skompiluj plik `Program.java`.
2. Uruchom skompilowany plik, aby zobaczyć informacje o figurach oraz obliczone sumy pól i obwodów.

## Dodatkowe Uwagi

- Projekt jest otwarty na rozbudowę o nowe figury geometryczne poprzez dodanie nowych klas dziedziczących po `Figura`.
- Zaleca się zapoznanie się z kodem każdej klasy, aby lepiej zrozumieć implementację poszczególnych figur.
